Viewport Units (`vw` and `vh`) in CSS

Header Section (Height: 40vh)

This section will always take up 40% of the viewport's height.

This text has a font-size of 5vw, making it scale directly with the viewport width.

This box attempts to be a square using 20vw for both width and height.

Image Gallery (Using vw for image width)

Each image is 25vw wide (adjusted for gap). They scale with viewport width.

This footer has a min-height of 10vh. It will be at least 10% of viewport height.

Content Box with Min/Max (`vw` + `clamp()`)

This box uses `clamp(300px, 60vw, 800px)` for its width. It means: min 300px, ideal 60% viewport width, max 800px. Try resizing your browser.
