Percentage (%) Unit in CSS

Width based on Parent

This is the parent div.

Child: 50% width of its parent.

Grandchild: 75% width of its parent (the child-width).

Parent with fixed width (400px).

Child: 50% width of its 400px parent.

Height based on Parent

This parent has a fixed height (200px).

Child: 50% height of its 200px parent.

This parent has auto height.

Child: 50% height. (Will NOT take 50% height here unless content defines it.)

Padding and Margin based on Parent Width

This parent has a specific width (400px).

Box with 5% padding (all sides relative to parent's 400px width).

Box with 10% margin (all sides relative to parent's 400px width).

Content inside aspect ratio box.

The empty `aspect-ratio-box` uses `padding-bottom: 56.25%;` to create a 16:9 aspect ratio.

Font Size based on Parent

Base font size parent (16px).

Child text: 120% of parent's font size.

Grandchild text: 120% of its parent's font size (which is already 120%).

Line Height based on Own Font Size

This paragraph has a large font size. Its line height is 150%, relative to its own font size.

This paragraph has a smaller font size. Its line height is also 150%, but it scales proportionally.
